# Security Policy

## Supported Versions

Currently, only the latest version of MeetBuddy AI is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within MeetBuddy AI, please do not open a public issue. Instead, send an email to giridhar2k20@gmail.com.

Please include:
- A description of the vulnerability.
- Steps to reproduce (if possible).
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
